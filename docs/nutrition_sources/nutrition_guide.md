# 中国居民膳食指南 (2022) 核心推荐

## 核心推荐一：食物多样，合理搭配
坚持谷类为主的平衡膳食模式。
- 每天的膳食应包括谷薯类、蔬菜水果、畜禽鱼蛋奶和豆类食物。
- 平均每天摄入12种以上食物，每周25种以上。
- 合理搭配。

## 核心推荐三：多吃蔬果、奶类、全谷、大豆
蔬菜水果、全谷物和奶类是平衡膳食的重要组成部分。
- 吃好晚餐：晚餐应清淡，以高蛋白质、低脂肪食材为主。
- 贫血调理建议：多食用鸭血、瘦肉、菠菜，并建议补充富含维生素C的食物以促进铁吸收。

## 食物成分参考
- 100克鸡胸肉含有约23.1克蛋白质。
- 一个中等大小的苹果（约200g）提供104 kcal热量。
